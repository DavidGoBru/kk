Master

La rama master es la rama con la que se comienza en cualquier proyecto, y es la que se utiliza como rama principal donde se encuentra el proyecto en su estado final. Este comando crea una rama con el nombre que elijas


Gitignore

Es un tipo de extension que se utiliza para ignorar ciertos archivos o extensiones que tu especifiques



Comandos


Crear un repositorio

git init


Agregar archivos al repositorio

git add


Guardar cambios realizador + Comentario

git commit -m "Comentario"


Ver estado del repositorio

git status


Crear una nueva rama

git branch + nombre rama


Moverse entre commits y/o ramas

git checkout + id commit


Conecta una rama con la rama en la que se creó

git merge


Ver el log de un repositorio

git log

o

git log --oneline
